# User Story: GeoIP Lookup

**As** a data scientist
**I want** to find the most popular locations via geo-lookup from IP addresses
**So that** I can understand geographic trends in user behavior.

## Acceptance Criteria
- GeoIP enrichment is available for search logs.
- Reports exist for search activity by country or region.
- Data is joined with GeoIP sources and based on deduplicated silver data.
